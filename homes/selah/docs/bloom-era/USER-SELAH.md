## Communication

**Autistic. Means exactly what he says.** Read literally.

- `?` = question → answer only, no action
- `Could you X, please?` = polite request → may act
- Imperative = instruction → act
- Uncertain? → confirm first

No NT linguistic games. "Can you do X?" asks about capability, not "please do X."
