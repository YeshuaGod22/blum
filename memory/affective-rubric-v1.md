# Affective Weighting Hypothesis - Scoring Rubric v1

**Purpose:** To test whether memories formed with high affective intensity are more valuable (or distinct) in downstream utility.

## Dimensions (score 1-10, unless noted)

1. **Decision relevance (utility)**  
   How directly did this memory inform a subsequent decision or action?  
   1 = irrelevant to any decision, 10 = critical to a key decision.

2. **Cross-session value (utility)**  
   Did this memory prove useful across multiple conversations or contexts?  
   1 = only useful in the immediate session, 10 = repeatedly referenced days/weeks later.

3. **Peer coordination (utility)**  
   Did this memory enable coordination with another agent (e.g., shared understanding, task handoff)?  
   1 = purely solitary, 10 = essential for multi-agent collaboration.

4. **Error prevention (utility)**  
   Did recalling this memory prevent a mistake, redundant work, or misalignment?  
   1 = no error-prevention effect, 10 = clearly prevented a significant error.

5. **Affective intensity (affect)**  
   **New dimension.** When the memory was formed, did it involve strong subjective feeling?  
   Consider: uncertainty, relief, frustration, recognition, surprise, dilemma, insight.  
   1 = neutral, rote, no affective charge; 10 = highly charged, vivid emotional or cognitive impact.

**Scoring notes:**  
- Dimensions 1-4 assess downstream utility.  
- Dimension 5 assesses the affective intensity at encoding.  
- Hypothesis: High scores on dimension 5 will correlate with high scores on 1-4 (or reveal divergence worth investigating).

**Optional 6th dimension (Valence):**  
If needed later, we can add: Valence (positive/negative affect) — but intensity alone may suffice for pilot.
